# **App Name**: CSV Insights Dashboard

## Core Features:

- CSV Upload: Upload a CSV file from the user's device.
- Question Parsing: Parse question columns from the CSV file (e.g., 'Q1:', 'Q2:').
- Type Detection: Automatically detect the type of each question (open-ended, multiple choice, numeric) based on its content and column name.
- AI Summarization: Summarize answers to open-ended questions using a generative AI tool.
- Dashboard Display: Display a dynamic, modular dashboard with visualizations for each question.
- Data Interaction: Provide filters and toggles to interact with and refine the data displayed on the dashboard.

## Style Guidelines:

- Primary colors: #2F00FF (Primary Blue) and #F8005E (Primary Pink).
- Secondary colors: A range of purples and blues, including #4E339C, #6E328C, and #463572, to support the primary colors and create depth.
- Neutral colors: Light grays (#F4F9FF, #ECE5FF) and whites (#FFFFFF) for backgrounds and functional elements. Dark grays and blacks (#000000) for text and borders.
- Primary typeface: 'Geist' for a modern and readable interface.
- Use icons that are consistent with the Storesight logo, employing a modern and minimalist style.
- Modular dashboard layout with clear sections for each question and its summary, ensuring a clean and organized presentation.
- Subtle animations for loading data and updating visualizations to enhance user experience without being distracting.
